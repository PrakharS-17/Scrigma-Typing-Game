# Scrigma-Typing-Game
SCRIGMA is a fast-paced typing speed game built with vanilla JavaScript, featuring dynamic difficulty, combo streaks, a timer system, and persistent high scores.

## Features
- Dynamic difficulty
- Timer with shrinking bar
- Score & streak system
- Pause / Resume
- Combo popup system
- LocalStorage high scores

## Controls
- Press ENTER to enter the word
- Press esc to toggle Pause/Resume
